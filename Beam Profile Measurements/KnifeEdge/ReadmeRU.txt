Данный скрипт является частью программы по автоматизации получения и обработки экспериментальных данных ANGEL.
Распространяется свободно для некоммерческих научно-исследовательских целей.
(c) Vulpes Corsac

Данный скрипт позволяет определить диаметр одномодового пучка и построить график каустики, при наличии более чем 3-х измерений.
Аппроксимация происходит с использованием следующих методов:
	1) P0 + P1/2 * (1 ± Erf[Sqrt[2](Px-P2)/P3]), где P0 - шумовая мощность, P1 - полная мощность излученя, P2 - координата середины пучка, P3 - радиус пучка по уровню E^(-2).
	2) y0 + A Exp[-(x-xc)^2/(2w^2)], где y0 - шумовая планка, A - амплитуда, xc - координаты середины пучка, w - половина раидуса пучка по уровню E^(-2).
Для аппроксимации Гауссом используется вычисление производной от исходных данных.

Данные на вход программы подаются в exel документе в следующем формате:
	1) Каждый эксперимент сохранён на отдельном листе
	2) Поле A1 соответствует расстоянию, на котором были произведены измерения
	3) Поля А2, В2 содержат наименования столбцов
	4) Далее в колонках А (положение лезвия) и В (измеренная мощность) идут данные
	5) Не должно быть никаких занятых ячеек на строчках ниже, чем закончились экспериментальные данные
	6) Такое ограничение на колонки не распространяется, т.е. в В можно записать среднее по всем колонкам правее её

Далее для получения обработанных данных необходимо:
	1) Поместить файл данных в ту же директорию, что и скрипт
	2) Открыть скрипт с помощью Wolfram Mathematica
	3) Выбрать название поддиректории, в которую будут сохранены все полученные данные и графики (По умолчанию dir = "Data"). Директория должна уже существовать
	4) Указать название файла (AllData = Import["data.xlsx", Path -> NotebookDirectory[]). Файл указывается с правильным расширением и обязан существовать. В принципе, поддерживаются форматы "*.xlsx", "*.xls".
	5) Выставить соответствующие настройки:
		а) Использовать ли планку мощности при аппроксимации
		б) Длина волны используемого излучения
	6) Нажать Evaluation->Evaluate NotebookDirectory
	7) Готово
